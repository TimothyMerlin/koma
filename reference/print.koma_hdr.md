# Print method for koma_hdr objects

Delegates to [`summary()`](https://rdrr.io/r/base/summary.html) for
console output.

## Usage

``` r
# S3 method for class 'koma_hdr'
print(x, ...)
```

## Arguments

- x:

  A `koma_hdr` object.

- ...:

  Additional arguments forwarded to
  [`summary.koma_hdr()`](https://timothymerlin.github.io/koma/reference/summary.koma_hdr.md).

## Value

Invisibly returns `x`.
