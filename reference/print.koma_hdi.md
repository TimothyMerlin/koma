# Print method for koma_hdi objects

Delegates to [`summary()`](https://rdrr.io/r/base/summary.html) for
console output.

## Usage

``` r
# S3 method for class 'koma_hdi'
print(x, ...)
```

## Arguments

- x:

  A `koma_hdi` object.

- ...:

  Additional arguments forwarded to
  [`summary.koma_hdi()`](https://timothymerlin.github.io/koma/reference/summary.koma_hdi.md).

## Value

Invisibly returns `x`.
