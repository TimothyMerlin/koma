# Get the type of a koma_ts object

Get the type of a koma_ts object

## Usage

``` r
type(x, type, var = NULL, ...)
```

## Arguments

- x:

  A koma_ts object.

- type:

  The type to filter by.

- var:

  An optional variable to filter by.

- ...:

  arguments passed to methods (unused for the default method).

## Value

A koma_ts object with the specified type.
