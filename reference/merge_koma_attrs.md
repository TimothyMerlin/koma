# Merge custom metadata from two koma_ts objects

Merge custom metadata from two koma_ts objects

## Usage

``` r
merge_koma_attrs(e1, e2 = NULL, op = NULL, template = NULL)
```

## Arguments

- e1:

  First operand.

- e2:

  Second operand.

- op:

  Name of the operation.

- template:

  Result template used for alignment.

## Value

A named list of merged custom attributes.
